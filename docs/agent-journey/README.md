# Agent Journey Documentation

This folder tracks the complete collaboration journey in sequence order.

This package is intended for interviewer review of Generative AI usage.

## Scope

- Feature-wise request and outcome trail
- Issues, root causes, and resolution prompts
- Interviewer-friendly Generative AI usage summary

## Mandatory Rules

- English only
- Sequence-only steps (`Step 1`, `Step 2`, ...)
- No date/time in entries
- Do not include original non-English prompts
- Include only refined professional prompt versions

## Standard Entry Format

Use this field order in all feature-step entries:

1. `Request`
2. `Action Taken`
3. `Result`
4. `Issue Encountered`
5. `Prompt Adjustment`
6. `Final Outcome`
